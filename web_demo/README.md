# 极简兵棋 - 网页演示版

一款类似《钢铁雄心》的极简版世界征服策略游戏。

## 🎮 如何运行

直接用浏览器打开此文件即可游玩：
```
C:\Users\Administrator\.gemini\antigravity\scratch\quest2_wargame\web_demo\index.html
```

或者双击 `index.html` 文件。

## 🎯 游戏规则

1. **目标**：消灭所有其他势力，成为最后的胜者
2. **领土**：地图上的圆圈代表领土，颜色代表所属势力
3. **兵力**：每个领土显示的数字是当前兵力，会随时间自动增长（最高100）
4. **攻击**：点击自己的领土选中，再点击相邻的敌方领土设为目标，调整派遣兵力后点击"发起攻击"
5. **战斗**：攻击时双方兵力相互抵消，剩余兵力多的一方获胜

## 🕹️ 操作说明

| 操作 | 说明 |
|------|------|
| 点击己方领土 | 选中该领土 |
| 点击敌方邻近领土 | 设为攻击目标 |
| 拖动滑块 | 调整派遣兵力 |
| 点击"发起攻击" | 执行攻击 |
| 点击"取消" | 取消选择 |

## ⚙️ 游戏设置

- **势力数量**：2-6 个势力可选
- **暂停/继续**：暂停或恢复游戏
- **加速**：3倍速度游戏
- **重开**：重新开始新游戏

## 🌍 世界地图

游戏包含 34 个领土，覆盖六大洲：
- 北美洲：阿拉斯加、加拿大、美国、墨西哥等
- 南美洲：巴西、安第斯、南锥体等
- 欧洲：西欧、中欧、北欧、俄罗斯等
- 亚洲：中国、日韩、东南亚、中东等
- 非洲：北非、西非、东非、南非
- 大洋洲：澳大利亚、太平洋岛屿
